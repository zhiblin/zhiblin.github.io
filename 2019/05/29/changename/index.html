Mac管理员账号丢失找回

事情起因是因为我想给我的管理员用户改名，结果改完之后用户的管理员权限没了。差点就要重装系统了。
还好在网上找到了这个方法。
暂定出问题的账号是 : UserA
暂定新建的账号是 : UserB

具体步骤如下:

  1. 进入单用户模式

命令是开机按住 cmd + s
这样会进入命令行界面

//对文件系统检查修复
fsck -y
//挂载分区
mount -uaw /
//删除系统安装完毕的配置
rm /var/db/.AppleSetupDone
//重启系统
reboot

2.新建管理员账号

重启之后进入系统，就会走一个新管理员用户新建的流程。新建的管理员账号UserB

3.把UserA设置为管理员

在系统偏好设置 -> 用户与群组 这个路径下
找到UserA，勾选 “允许用户管理这台电脑”
这样UserA就回归之前的管理员账号了。

重启一下进入UserA用户

为了安全最好还是把UserB改成普通号。
或者直接删除
