Mac自己签名软件，防止闪退

Aug 5, 2019

安装Xcode

安装Xcode，你可以在App Store中下载安装，并且至少运行一次。

安装Command Line Tools 工具
打开终端工具输入如下命令：

xcode-select –install

弹出后选择继续安装。

签名

打开终端工具输入并执行如下命令：

codesign --force --deep --sign - /Applications/AppName.app

注意(/Applications/AppName.app)文件路径，你可以打开访达找到应用程序，找到要签名的软件，直接拖入 终端 界面，即可自动生成路径。
