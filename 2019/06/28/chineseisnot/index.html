String识别是否包含中文

Jun 28, 2019

使用Unicode编码表汉字对应的编码范围。
对比字符串中是否这个范围的编码，从而判断是否包含汉字。

func isIncludeNotChineseIn(string: String) -> Bool {
    for (_, value) in string.enumerated() {
        if !("\u{4E00}" <= value && value <= "\u{9FA5}") {
            return true
        }
    }
    return false
}
